# BookNest プライバシーポリシー雛形

最終更新日: 2026年7月10日

この文書は公開前に内容を確認し、運営者名、問い合わせ先、公開URLを実際の情報へ差し替えてください。必要に応じて法律・規約の専門家による確認を受けてください。

## 1. 取得する情報

BookNest（以下「本アプリ」）は、以下の情報を取得または保存する場合があります。

- アカウント情報: メールアドレス、認証ID
- 蔵書情報: 書籍名、シリーズ名、巻数、ISBN、著者、出版社、表紙画像URL、読書ステータス、登録日時
- 通知情報: Expo Push Token、新刊通知のON/OFF、シリーズごとの通知設定、通知送信ログ
- アプリ設定: テーマ、表示設定、外部リンクの開き方など
- 問い合わせ時にユーザーが任意で提供する情報

## 2. 利用目的

取得した情報は、以下の目的で利用します。

- 蔵書・読書状況を保存、同期、表示するため
- ISBN検索、表紙画像取得、シリーズ情報取得を行うため
- 新刊通知や通知テストを送信するため
- アカウント管理、ログイン、ログアウト、アカウント削除を行うため
- 不具合調査、品質改善、問い合わせ対応を行うため

## 3. 外部サービスの利用

本アプリは、機能提供のために以下の外部サービスを利用する場合があります。

- Supabase: 認証、データベース、Edge Functions
- Expo Push Notifications: プッシュ通知配信
- OpenBD API: 書籍情報取得
- Google Books API: 書籍情報取得
- Rakuten Books API: 書籍情報取得
- 外部ECサイトまたはブラウザ: 購入候補ページの表示

外部サービスでは、各サービスのプライバシーポリシーや利用規約に従って情報が取り扱われます。

## 4. 第三者提供

運営者は、法令に基づく場合を除き、ユーザーの個人情報を本人の同意なく第三者へ販売または提供しません。ただし、本アプリの機能提供に必要な範囲で、上記の外部サービスに情報が送信される場合があります。

## 5. データの保存期間と削除

ユーザーは、アプリ内の設定画面からアカウント削除を実行できます。アカウント削除を行うと、クラウド上に保存された蔵書情報、通知トークン、シリーズ通知設定、通知ログ、認証アカウントが削除されます。

端末内に残る一時データ、エクスポート済みファイル、外部サービス側で保持されるログやバックアップについては、各環境または各サービスの仕様に従います。

## 6. 安全管理

運営者は、取得した情報の漏えい、滅失、改ざんを防ぐため、アクセス制御、認証、通信の暗号化など、合理的な安全管理措置を講じます。

## 7. 未成年の利用

未成年のユーザーが本アプリを利用する場合は、保護者の同意を得たうえで利用してください。

## 8. プライバシーポリシーの変更

運営者は、必要に応じて本ポリシーを変更することがあります。重要な変更がある場合は、アプリ内または公開ページで通知します。
